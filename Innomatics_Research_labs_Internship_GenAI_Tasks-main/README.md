### 1. Token Classification (POS Tagging & Chunking)
**Folder/File:** `Task5_Token_Classification.ipynb`

* **Dataset:** CoNLL-2003 (subset for local processing)
* **Model:** `distilbert-base-uncased`
* **Tasks Achieved:**

    * Evaluated sequence accuracy using the `seqeval` library to calculate precision, recall, and F1 scores.

### 3. Mastering LangChain Prompt Templates
**Folder/File:** `Task1_GenAI_Prompt_Templates.ipynb`

This assignment marked the transition from traditional NLP to Generative AI. The objective was to shift from hardcoded, static prompts to dynamic, modular, and reusable prompt systems using the LangChain framework.

* **Frameworks Used:** `langchain_core`
